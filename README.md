
# TechSpree Task

### Clone the repository


```bash
  git clone https://github.com/Vaibhavsun/TechSpree-Task.git
```
```bash
  cd TechSpree-Task
```

### creating Virtual environment

```bash
  python -m venv env
```

### installing Requirements

```bash
  pip install -r Requirements.txt
```

### Run Flask on local server

```bash
  python main.py
```
